# Movie and TV Recommendation Test Project

This repository is a small personal project used for local development and testing of movie and TV discovery ideas.

## Overview

The goal of this project is to explore a simple recommendation-style interface for browsing movies and series by user preference. It focuses on lightweight frontend testing, UI structure, and basic filtering behavior during early development.

## Current scope

- Simple static web interface
- Movie and series cards
- Basic client-side filtering
- Prototype-friendly structure for future metadata integration

## Planned development

- Add live metadata integration
- Improve recommendation logic
- Expand filters for mood, genre, and watch preferences
- Test layout ideas for a small discovery experience

## Status

This is an early-stage local test project intended for development and experimentation.
